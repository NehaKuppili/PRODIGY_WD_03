# 🎮 Novaris XO Arena

A modern and interactive Tic-Tac-Toe web application built using HTML, CSS, and JavaScript as part of the Prodigy InfoTech Web Development Internship.

## Features

- Interactive 3×3 Tic-Tac-Toe board
- Player X and Player O turns
- Winner detection
- Draw detection
- Score tracking
- New Game functionality
- Responsive design
- Playful and modern UI

## Technologies Used

- HTML
- CSS
- JavaScript
